Thank you for considering a contribution to the Bpod project!

To contribute, 
1. Make a fork of Bpod_Gen2.
2. Clone your fork, make updates, and commit the updates to your fork.
3. Submit a pull request for us to integrate your changes into the project.

***IMPORTANT***
We kindly request that you submit your pull request against the 'develop' branch of Bpod_Gen2, so that we can test the code before it is added to the default software for everyone. Pull requests made against 'master' will be closed, with a request to resubmit against 'develop'. Thank you!
